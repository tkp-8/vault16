Definition

$$ | a| = \begin{cases} a, &\text{ falls }a\ge 0\\ -a, &\text{ falls } a<0 \end{cases}

$$

Eigenschaften

Fuer alle a,b\in \mathbb{R} gilt:

  1. |ab|=|a| |b|
  2. |a+b|\le |a|+|b|
  3. |\frac{a}{b}|=\frac{|a|}{|b|}, falls b\neq 0
  4. ||a|-|b||\le \begin{cases}|a-b| &\\|a+b|\end{cases}
