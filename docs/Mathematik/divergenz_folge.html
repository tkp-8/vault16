Definition

Eine reelle Folge heisst divergent, wenn sie nicht konvergent ist

Eigenschaften

Proposition

Genau dann konvergiert (a_n) nicht gegen a, wenn es eine \varepsilon_{0}-Umgebung von a so gibt, dass eine Teilfolge von (a_n) vollstaendig ausserhalb von U_{\varepsilon_{0}}(a) liegt

Beispiele

  • (n)
  • ((-1)^{n})
  • (a_n)=(\sum_{i=1}^{n}\frac{1}{i})
  • (a_n)=((-1)^{n+1}+\frac{1}{n})
  • (q^{n}) mit |q|>1
