Mersenne’sche Primzahlen

2025-05-04 09:46

Tags:

Definition

Die n-te Mersenne’sche Zahl M_n hat die form M_n=2^n-1 mit n\in \mathbb{N}. Ist M_n eine Primzahl, so wird sie Mersenne’sche Primzahl genannt.

Eigenschaften

Seien a,n\in \mathbb{N} mit n\ge 2

Proposition 1: Wenn a^{n}-1 eine Primzahl ist, dann ist a=2.
Beweis: a^{n}-1 = (a-1)(a^{n-1} + a^{n-2} + \cdots + a+1) Wenn a^{n-1} eine Primzahl ist, dann muss der Teiler a-1 von a^{n}-1 gerade 1, also a=2 sein.
