Definition

Wenn L eine Sprache ist, fuer die es eine kontextfreie Grammatik G gibt, sodass L(G)=L, dann nennen wir L eine kontextfreie Sprache. CFL \coloneqq \{L\mid L \text{ ist kontextfrei}\}
